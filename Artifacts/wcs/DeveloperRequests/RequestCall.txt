   "orderItem": [

    "productId": "10641",
    "quantity": "2.0"
  }
   ]
}
	  
Get Shopping Cart:
Request : GET :http://localhost:5000/wcs/cart/shoppingCart?userId=9014


Update Shopping Cart Items:
Request : PUT :localhost:5000/wcs/cart/updateShoppingCartItem?userId=9014
Input Body : 
 {
   "lineItem": [
      {
         "lineItemId": "80072",
           "quantity": "5"
       }
   ] 
}

Request: DELETE :localhost:5000/wcs/cart/deleteShoppingCartItem?userId=9014

  }
   ] 
}

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
                                        WISHLIST
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Add Item To WishList: 

Request: POST: http://localhost:5000/wcs/wishlist/addToWishList?userId=9018

Request - Body: 
{
  "productId": "10140",
  "quantityRequested": "1"
}

Response: 

{
    "success": true
}

Delete an item from WishList: 

Request: DELETE: http://localhost:5000/wcs/wishlist/deleteFromWishList?wishListId=12526&wishListItemId=13039&userId=9018

If wishListItemId is not specified in the request, the entire wishlist will be deleted. 
Specify WishListItemId for deleting specific item from the wish list.

Response: 

{
    "success": true
}

Get WishList: 

Request: GET : http://localhost:5000/wcs/wishlist/getWishList?userId=9018
