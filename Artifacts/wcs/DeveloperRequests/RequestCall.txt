Add Item To WishList:

Request - Body: 
{
  "productId": "10140",
  "quantityRequested": "1"
}

Response: 

{
    "success": true
}

Delete an item from WishList: 

Request: DELETE: http://localhost:5000/wcs/wishlist/deleteFromWishList

If wishListItemId is not specified in the request, the entire wishlist will be deleted. 
Specify WishListItemId for deleting specific item from the wish list.

Request Body: 

{
  "wishListId": "12542",
        "itemList": [
            {
                "wishListItemId": "13059"
            }
        ]
}

Response: 

{
    "success": true
}
