++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++


Add to Cart :
Request : POST : http://localhost:5000/wcs/cart/addToCart?userId=9011
Input Body : 
{
   "orderItem": [
      {
    "productId": "10641",
    "quantity": "2.0"
  }
   ]
}
	  
Get Shopping Cart:
Request : GET :http://localhost:5000/wcs/cart/shoppingCart?userId=9014


Update Shopping Cart Items:
Request : PUT :localhost:5000/wcs/cart/updateShoppingCartItem?userId=9014
Input Body : 
 {
   "lineItem": [
      {
         "lineItemId": "80072",
           "quantity": "5"
       }
   ] 
}
 

Delete Shopping Cart Item:
Request: DELETE :localhost:5000/wcs/cart/deleteShoppingCartItem?userId=9014
Input Body:
{
   "lineItem": [
      {
         "lineItemId": "80071",
          "quantity": "0"
  }
   ] 
}

Delete All Shopping Cart Item:
Request: DELETE : localhost:5000/wcs/cart/deleteAllShoppingCartItem?userId=9014
